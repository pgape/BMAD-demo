# 麻将游戏项目简介

## 项目名称
四人麻将游戏核心系统

## 项目背景
麻将是中国传统棋牌游戏，具有广泛的用户基础。本项目旨在开发一个数字版的四人麻将游戏，验证核心游戏流程的可行性。

## 项目目标
开发一个可运行的四人麻将游戏原型，验证核心游戏流程的可行性，包括完整的牌局设置、游戏循环、响应处理和胜负判定。

## 功能范围

### 核心功能
1. **基础牌局设置**
   - 136张标准麻将牌（不含花牌）
   - 洗牌和砌墙
   - 发牌给4位玩家

2. **游戏主循环**
   - 摸牌、打牌机制
   - 玩家操作（暗杠）
   - 响应处理（吃、碰、杠、胡）

3. **基础胡牌评估**
   - 实现最基础的胡牌牌型判断
   - 支持平胡等简单牌型

4. **状态管理**
   - 跟踪游戏状态、玩家手牌、牌墙剩余数

### 初始版本限制
- 不包含花牌机制
- 不包含复杂的番种计算
- 不包含AI玩家，仅支持真人玩家

## 项目约束
- 需要在有限时间内验证核心流程
- 重点关注游戏逻辑的正确性
- 用户界面可以保持简单

## 成功标准
- 4名玩家可以完成一整局游戏
- 核心游戏机制正常运行
- 胡牌判断准确
- 响应处理符合规则

## 预期成果
一个可运行的麻将游戏原型，验证核心游戏流程的可行性，为后续完整开发奠定基础。
